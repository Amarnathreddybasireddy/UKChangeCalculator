# UK Currency Change Calculator

A lightweight, robust .NET 10 console application designed to calculate and break down change due to a customer in standard UK sterling (£) denominations, sorted from largest to smallest.

## Features
* Handles input values safely to prevent application crashes.
* Converts inputs to integer pence to eliminate floating-point arithmetic errors.
* Displays outputs clearly using standard UK denomination symbols (e.g., £10, £2, 50p).

## How to Build and Run

### Prerequisites
* [.NET 10.0 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) or higher.

### Run via Command Line
1. Open your terminal or command prompt.
2. Navigate to the project folder: cd UKChangeCalculator
3. Run the application using the following command: dotnet run
